# 🌱 AgroMIP

## Manejo Integrado de Pragas para uma Agricultura Sustentável

O **AgroMIP** é um projeto educacional desenvolvido para o **Programa Agrinho 2026**, com o objetivo de apresentar de forma interativa os conceitos e benefícios do **Manejo Integrado de Pragas (MIP)**.

A plataforma busca conscientizar estudantes, produtores rurais e a comunidade sobre a importância de práticas agrícolas sustentáveis que aumentam a produtividade, preservam o meio ambiente e fortalecem a conexão entre o campo e a cidade.

---

## 🎯 Objetivos

* Explicar os fundamentos do Manejo Integrado de Pragas.
* Demonstrar a importância do monitoramento das lavouras.
* Incentivar o uso de controles biológicos.
* Reduzir a dependência de defensivos químicos.
* Promover a sustentabilidade na agricultura.
* Contribuir para a educação ambiental e agrícola.

---

## 🌾 O que é o MIP?

O Manejo Integrado de Pragas é uma estratégia que combina diferentes métodos de controle para manter as populações de pragas abaixo do nível de dano econômico.

Entre as principais técnicas utilizadas estão:

* Monitoramento constante da lavoura;
* Controle biológico;
* Controle cultural;
* Controle mecânico;
* Uso responsável de defensivos agrícolas quando necessário.

## 🌎 Relação com o Agrinho 2026

O projeto está alinhado aos princípios do Agrinho ao demonstrar como a agricultura sustentável beneficia toda a sociedade.

O Manejo Integrado de Pragas contribui para:

* Produção de alimentos mais seguros;
* Preservação dos recursos naturais;
* Redução de impactos ambientais;
* Valorização do trabalho no campo;
* Fortalecimento da conexão campo-cidade.
